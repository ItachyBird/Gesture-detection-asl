    • Gesture Recognition

    1. Import Libraries and Set Up Environment: Import necessary libraries like OpenCV, MediaPipe, and others required for gesture recognition.
    2. Initialize Parameters: Set parameters for the collection such as the action being performed, starting folder, number of sequences (videos), and the length of each sequence (number of frames).
    3. Capture Video: Open the webcam for capturing video frames.
    4. Set MediaPipe Model: Initialize the MediaPipe Holistic model for detecting and tracking holistic landmarks.
    5. Collect Data:
      • Loop through sequences and frames.
      • Capture each frame from the video.
      • Process the frame using MediaPipe to get landmark results.
      • Annotate the frame with landmarks and additional information.
      • Save the extracted keypoints from the landmarks into a .npy file.
    6. Display Frame: Show the annotated frame in a window.
    7. Terminate Capture: Release the video capture and destroy all OpenCV windows upon completion or interruption.
